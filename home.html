Lezione sulle Funzioni in C#

Impara a creare e utilizzare le funzioni con un tocco di magia Disney!

Introduzione alle Funzioni

Le funzioni sono blocchi di codice che eseguono istruzioni. Sono come delle “ricette” magiche che possiamo riutilizzare.

Esempio di Funzione Semplice


using System;

class Program
{
    static void Saluta()
    {
        Console.WriteLine("Ciao, benvenuto!");
    }
    
    static void Main(string[] args)
    {
        Saluta();
    }
}

Funzione con Parametri


using System;

class Program
{
    static void Saluta(string nome)
    {
        Console.WriteLine("Ciao, " + nome + "!");
    }
    
    static void Main(string[] args)
    {
        Saluta("Marco");
    }
}

Esercizio: Somma di Due Numeri


using System;

class Program
{
    static int Somma(int a, int b)
    {
        return a + b;
    }
    
    static void Main(string[] args)
    {
        int risultato = Somma(3, 4);
        Console.WriteLine("La somma è: " + risultato);
    }
}
